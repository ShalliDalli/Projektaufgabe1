VOIP (Voice Over IP)

Mithilfe von VOIP ist es möglich Telefonate über das Internet führen zu können.

VOIP bietet somit eine Resourcenschonendere Möglichkeit für das Telefonieren an.
SIP (Session Initialisation Protocol)

Mithilfe des SIP kann eine Kommunikationsverbindung zwischen mehreren Teilnehmern auf- und abgebaut werden.
RTP (Real-Time Transport Protocol)

Mithilfe von RTP können Multimedia-Inhalte als Pakete über das Internet übertragen werden.

RTP ist mit SIP ein grundlegender Bestandteil von VOIP.
sip.conf

[general]
language=de
context=default
bindport=5060 ;Port der für SIP verwendet wird

disallow=all;Standardmäßig werden alle Codecs nicht erlaubt
